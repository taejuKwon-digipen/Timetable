# 시간표 관리 프로그램

Python/PyQt6로 만든 학생 수업 시간표 자동 생성 및 편집 프로그램입니다.

## 기능

- **학생 추가**: 이름, 나이, 전화번호, 주소, 수업진행시간, 불가능한 시간대 입력
- **자동 시간표 생성**: 불가능한 시간대와 나이(중학생 16시, 고등학생 18시 이후)를 고려해 배정
- **7요일 지원**: 월~일 전부 수업 고려
- **블록 드래그앤드롭**: 생성된 시간표에서 블록을 드래그해 원하는 시간대로 이동
- **학생 정보 표시**: 각 수업 블록에 이름·주소 표시

## 설치 및 실행

```bash
pip install PyQt6
python main.py
```

## 사용 방법

1. **+ 학생 추가** 버튼으로 학생 등록
2. 각 학생의 불가능한 시간대를 "시간대 추가"로 입력
3. **시간표 짜기** 버튼 클릭
4. 생성된 시간표에서 블록을 드래그해 원하는 칸으로 이동
